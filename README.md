# Obfuscator

Welcome to the Obfuscator project! This README will guide you through the documentation and usage of the project.

## Overview
The Obfuscator is a tool designed to obfuscate code in various programming languages, making it difficult to read while maintaining functionality.

## Features
- **Supports multiple languages:** The tool supports various programming languages, including JavaScript, Python, and PHP.
- **Customizable options:** Users can customize the level of obfuscation based on their needs.
- **Easy integration:** The tool can be easily integrated into existing build pipelines.

## Installation
To install the Obfuscator, clone the repository and install the dependencies:

```bash
git clone https://github.com/xrimce/Obfuscator.git
cd Obfuscator
npm install
```

## Usage
To obfuscate a file, run the following command:

```bash
node obfuscator.js yourfile.js
```

## Configuration
You can create a configuration file named `config.json` to specify your options:

```json
{
  "language": "JavaScript",
  "level": "high"
}
```
